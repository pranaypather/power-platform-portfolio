# Security Incident Reporting App

## Business Problem
Employees lacked a simple method to report suspected phishing and security incidents,
leading to delayed responses and increased risk.

## Solution
A Power Apps reporting app integrated with Power Automate to log, notify,
and track security incidents.

## Features
- Incident reporting form
- Severity classification
- Automated alerts to IT/Security
- Incident log and reporting
- Monthly summary automation

## Tools Used
- Power Apps
- Power Automate
- SharePoint / Dataverse
- Outlook

## Outcome
Improved security awareness, faster incident response, and better visibility
into security trends.
