# Employee Onboarding Automation

## Business Problem
Employee onboarding required multiple manual steps across departments,
resulting in delays, missed tasks, and inconsistent communication.

## Solution
An automated onboarding workflow using Power Automate triggered by a form submission,
coordinating HR, IT, and management notifications.

## Features
- New hire data capture
- Approval workflow
- Automated welcome email
- HR and IT task notifications
- Audit trail of onboarding steps

## Tools Used
- Power Automate
- Microsoft Forms
- SharePoint
- Outlook
- Microsoft Teams

## Outcome
Reduced onboarding time, improved accountability, and consistent onboarding
communication across departments.
