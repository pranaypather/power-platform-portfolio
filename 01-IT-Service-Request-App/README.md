# IT Service Request App

## Business Problem
IT support requests were submitted via email with no tracking, prioritisation,
or visibility, leading to delayed response times and lost requests.

## Solution
A Power Apps Canvas App integrated with Power Automate to capture, track,
and manage IT service requests from submission to resolution.

## Features
- Request submission with category and priority
- Automatic email and Teams notifications
- Status tracking (New, In Progress, Resolved)
- Centralised request log
- Basic SLA prioritisation logic

## Tools Used
- Power Apps
- Power Automate
- SharePoint
- Outlook
- Microsoft Teams

## Outcome
Improved response tracking, reduced manual email handling, and better visibility
into IT workload and performance.
